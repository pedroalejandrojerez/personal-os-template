# Review Week

Run the weekly operating-system review.

Inputs:

- `memory/pending-improvements.md`, if present
- Recent git activity
- `handoff.md`
- `active-backlog.md`
- Existing feedback memories
- Existing hooks and workflow checks

Audit workflow adoption:

- Task lanes used.
- Source of truth named.
- QA checklists used.
- Pre-ship checks run.
- Closeout receipts complete.
- Short approvals kept narrow.

Recommend one durable artifact when a repeated pattern is found.
