# Compound Engineering

Run the full build workflow.

Use when the user wants to move from idea to shipped work with planning, execution, review, and learning.

Workflow:

1. Check whether something already built can ship first.
2. Clarify the outcome and smallest useful version.
3. Route to G-Stack planning when the work is non-trivial.
4. Use `/grill-me`, `/build-prd`, and `/prd-to-slices` for larger builds.
5. Let Composer or Codex execute approved slices.
6. Run review, QA, and pre-ship verification.
7. Capture reusable lessons only when useful.
